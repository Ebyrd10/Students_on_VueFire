# Students_on_VueFire
A small app to display a table of imaginary students, stored in a cloud database, with CRUD functionality

## Technology
	
* HTML
* JavaScript
* CSS
* Vue
* VueCaptcha
* Google reCaptcha
* Cloud Firestore
* VueFire
* Vuetify
* Cloud Functions

Front-End

![students_vuefire](students_vuefire.png)


Database on Firestore

![firestore_db](firestore_db.PNG)

## Install

```sh
npm install
```

## Usage

```sh
npm run serve
```
